# Uncertainty-Aware Liver Cirrhosis Classification under Noisy and Outlier-Rich Clinical Data

## Overview
This project focuses on developing robust machine learning models for liver disease classification under noisy and outlier-rich clinical data conditions. While existing approaches achieve high predictive accuracy on clean datasets, real-world medical data is often imperfect due to measurement errors, missing values, and external disturbances.

The primary objective of this work is not only to achieve strong predictive performance but also to evaluate the reliability, stability, and confidence of model predictions under realistic data perturbations using uncertainty estimation techniques.

---

## Problem Statement
Most existing machine learning models for liver disease prediction assume that the available dataset is clean and representative of real-world conditions. However, clinical data collected in practice is often affected by noise, outliers, and inconsistencies arising from instrumentation errors, human factors, and data acquisition limitations.

This project investigates how the presence of such imperfections impacts model performance and whether uncertainty-aware approaches can maintain reliable predictions even under degraded data conditions.

---

## Research Gap
Existing literature on liver disease prediction primarily focuses on improving model accuracy through better preprocessing, feature engineering, and model selection. While these methods achieve strong results on benchmark datasets, they do not address a critical real-world concern:

**How stable and trustworthy are these models when the input data is noisy or contains outliers?**

Most studies:
- Assume clean and well-preprocessed datasets  
- Do not evaluate robustness under controlled noise conditions  
- Do not analyze prediction consistency under perturbations  
- Do not incorporate uncertainty estimation to assess model confidence  

This project aims to bridge this gap by systematically introducing controlled noise and outliers into the dataset and evaluating model robustness alongside uncertainty-aware predictions.

---

## Dataset
This project uses the **Indian Liver Patient Dataset (ILPD)** from the UCI Machine Learning Repository:

https://archive.ics.uci.edu/dataset/225/ilpd+indian+liver+patient+dataset

The dataset contains clinical and pathological attributes such as bilirubin levels, enzyme measurements, and protein indicators, which are used to classify liver disease presence.

---

## Objectives
- Develop classification models for liver disease prediction  
- Introduce controlled noise and outlier conditions into clinical data  
- Evaluate model robustness under degraded data conditions  
- Analyze prediction stability and consistency  
- Incorporate uncertainty estimation to assess model confidence
